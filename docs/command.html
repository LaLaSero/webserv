サンプルコマンドリスト

以下のサンプルコマンドでテスト項目を試してみてください

必要であればループバックアドレスの設定

sudo ip addr add 127.0.0.2/8 dev lo

GETメソッドの確認

curl -i localhost:8081/docs

POSTメソッドによるファイルのアップロードの確認

curl -i -X POST -F upfile=@./test.py localhost:8081/docs/upload/

アップロードしたファイルの確認

curl -i localhost:8081/docs/upload/posted.txt

DELETEメソッドの確認

curl -v localhost:8081/docs/upload/posted.txt -X DELETE

不適切なメソッドの確認

curl -X FOO localhost:8081/docs

ボディサイズを超えるリクエスト

curl -X POST -H "Content-Type: plain/text" --data "BODY IS HERE write something short or long than body limit" localhost:8081/docs/upload/

AutoIndexの確認

curl -i localhost:8081/docs/

CGIの確認

curl -i "localhost:8081/docs/cgi-bin/test.py?type=document-response"

siegeコマンドによる負荷テスト

siege -b http://localhost:8081/docs/
