サンプルコマンドリスト

以下のサンプルコマンドでテスト項目を試してみてください

GETメソッドの確認

curl -i localhost:8081/docs

POSTメソッドによるファイルのアップロードの確認

curl -i -X POST -F upfile=@./test.py localhost:8081/docs/upload/

DELETEメソッドの確認

curl -v localhost:8081/docs/delete/a.txt -X DELETE

AutoIndexの確認

curl -i localhost:8081/docs/index/

CGIの確認

curl -i localhost:8081/docs/cgi-bin/test.py

同じポートで違う名前のサーバー

curl --resolve defaultserver:2020:0.0.0.0 http://defaultserver:2020/hello/

curl --resolve virtualserver:2020:0.0.0.0 http://virtualserver:2020/hello/
